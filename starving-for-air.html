Starving for Air

A game Developed in RenPy Engine as a personal project.

The project took 2 months to develop. Featuring my friends, this game quickly became an inside joke within my friend group. The complex part of this project is drawing each character sprite with a different state, keeping a specific artstyle at the same time.

The goal of the game is to survive weird and random situations, making choices which define the ending.
