Countries

An application developed using React.js and Countries API as a college project.

The project took 3 weeks to develop. Using React.js to optimise the project into one page was helpful for viewing, and Bootstrap was a great addition to improve the visual aspect.

The API is used to display every country in the world, its information and location on the map.
